# Quality Checklist

Run this before delivery.

## Narrative

- Each slide has one main job.
- The first slide makes the topic and point obvious.
- The last slide gives a clear takeaway or next action.
- The deck has rhythm changes, not repeated card grids.
- Dense source material has been summarized, not pasted.

## Design

- The style matches the topic and audience.
- Text hierarchy is clear.
- Margins and alignment are consistent.
- Chinese text does not overflow or wrap awkwardly.
- There are no generic emoji bullets.
- There is no default purple-blue AI gradient unless it is intentional.
- Images have information value.
- Each visual page has search keywords or a clear no-search reason.
- Real/generative image candidates passed relevance and quality checks before use.
- Selected images match the page layout role instead of being direct search-result inserts.
- Mascots, products, packages, red envelopes, and symbolic objects are cut out when a transparent subject would improve the layout.
- Diagrams are understandable without narration.

## Editability

- Main titles are editable.
- Body copy is editable.
- Diagram labels are editable when practical.
- Flattened elements are acceptable and disclosed.
- PPTX page count matches the source.

## Export

- HTML opens correctly.
- PDF page count is correct.
- PPTX opens correctly, if generated.
- Screenshots/contact sheet show no blank slides.
- No visible overlap, clipping, or bottom-safe-area collisions.

## Delivery

- Provide exact output paths.
- State what is editable.
- State what is flattened.
- Mention any assumptions or missing assets.
